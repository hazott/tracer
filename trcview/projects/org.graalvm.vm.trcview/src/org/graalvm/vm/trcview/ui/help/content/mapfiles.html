Map Files

TRCView can import two different types of map files: binutils map files and map files generated by IDA Pro.

Binutils Map Files

To quickly import symbols from an ELF file, you can load a binutils map file. Such a map file can be generated with the following command:

nm program.elf > program.map

Keep in mind that the program might have been relocated during execution. If the program was relocated, you will have to adjust the addresses in the map file.

Binutils Map File Format

Every line defines a symbol and consists of three fields separated by space. The first field is the address in hexadecimal notation, the second field is the symbol type and the third field is the symbol name. Only symbols with type a, A, t, and T are processed.

Example:

00011a18 T usage

This line defines a symbol at address 0x00011a18 (T means "text", so this is probably a subroutine) with the name usage.
