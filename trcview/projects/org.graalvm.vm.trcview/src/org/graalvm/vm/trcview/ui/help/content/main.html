Main Window

The main window shows the execution trace as well as all other relevant information. You will probably spend most of your time here.

Key Bindings

Key Action
nrename symbol
yset function type
/set comment (current timestamp)
;set comment (memory location)
@set dynamic comment
hset color (memory location)
ggo to program counter
igo to timestamp
c"continue" to next execution of current instruction
<C-O>open trace file
<C-L>load session file
<C-S>save session file
<C-Q>exit TRCView
<CR>step into subroutine
<ESC>return from subroutine
<S-F2>open ABI editor
<S-F5>open data view
<S-F6>open device view
